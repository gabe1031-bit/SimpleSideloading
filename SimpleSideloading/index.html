Simple Sideloading

Welcome to Simple Sideloading!

This project is dedicated to the plan to make sideloading easy for legacy and any iOS device. Most websites already do this, but we focus on simplistic UI and little to no JavaScript for legacy devices to easily log on and start sideloading.

Check out our discord on a newer device for announcements and fixes.

If you have trouble downloading/installing any of the apps below, you may need an SSL Fix (Root Certificate Update). To get this, install this certificate . If you end up downloading a .pem instead of a popup, try vistiting this site on Safari.
